😰 肝度分析

玩家普遍感到肝度较高，尤其在资源获取和活动安排上存在明显疲劳感，但不同服表现有差异。

  • 1区: 第[2, 3, 4]天出现, 峰值第2天 — 资源获取和BOSS难度导致持续疲劳
  • 2区: 第[1, 2, 3]天出现, 峰值第2天 — 突破材料和抽卡爆率引发高疲劳
  • 3区: 第[1, 2]天出现, 峰值第1天 — 新手资源管理困难但疲劳度较低
