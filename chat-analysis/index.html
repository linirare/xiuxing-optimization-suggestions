一、肝度问题：好累啊，太肝了

🥵 肝度/累（玩家说"好累啊"就是"太肝了"，合并为一个指标）
126条
阈值≥0.60 · 占总量1.28%

核心是"累"质心匹配126条，其中明确说"肝/肝不动"的有5条。语义方向不同不合并质心，但标签合并——玩家表达的都是"这游戏让人累"。

结论： 肝度/累共126条（1.28%），峰值在2服D4=1.8%和3服D2=2.4%。
明确提到"肝"的只有5条（0.05%），但玩家说"太肝了"和"好累啊"是一回事——游戏玩着累。
不是核心矛盾（远低于付费16%和战斗22%）。

🎯 运营建议（量化依据）

🟥 紧急
1. 新手期玩法重构 — D1无聊率3.0-3.5%，玩家开服当天就觉得"点点点没意思"。依据：2服D1无聊3.5%=60条/579条
2. 2服付费口碑干预 — 2服D1付费不满8.6%+讨论9.5%=18.1%，93003/93005渠道对价格敏感。依据：2服D1付费不满8.6%是3服4.5%的1.9倍
🟧 重要
3. 暗号系统引导 — 3服D1 6.5%的人在问暗号。一个对话NPC可解决。
4. 3服D2内容填充 — 3服D2无聊率翻倍到4.0%，核心是"人少+内容消耗快"。依据：3服D2 4.0% = 13/329条
5. 操作优化 — 1服D1"按起来好累"，核心操作体验受诟病。
6. 社交引导分层 — 1服势力频道活跃（44.4%）但2/3服几乎不用。2服设计势力频道入口。
🟢 一般
7. 肝度不是核心矛盾 — 肝度/累共126条（1.28%），其中明确说"肝"的只有5条。
8. BUG修两个核心 — "进不去"和"卡死"占了大部分技术反馈。

⚙️ 方法说明

检索引擎MiniMax embo-01 文本嵌入模型，1536维，同空间(type=query)余弦相似度
分析方式每个维度10-11条语义查询句 → 质心向量 → 逐条计算相似度
阈值自适应：优先≥0.55，匹配不足时降级至0.50→0.45
数据清洗排除93010渠道 → 排除地块/兵器/灵兽/防具分享 → 排除论道/切磋 → 去重(同服+同消息+同分钟) → 排除广告 → 排除纯表情
