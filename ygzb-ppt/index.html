Deep Integration · Option A

末日废土深度融合

开发成本 中 推荐指数 ★★★★★

狱国 → 末日映射

狱国原版末日废土包装
出兵攻打城池→派出搜尸队前往废墟/僵尸区
兵种（青州兵/大象）→幸存者职业：搜刮者·枪手·掷弹兵·重盾
城防/建筑升级→庇护所升级：工坊·医疗·兵营·雷达
资源（金币/木材）→废土资源：水·食物·弹药·零件
联盟国战→营地间资源争夺战
武将/英雄→幸存者领袖：医疗兵·机械师

美术方向

风格 Q版暗黑卡通（避开无尽冬日写实风）

参考 幸存者33天 Q版僵尸 + 辐射避难所明亮废土

色板 暖黄/橙 + 冷灰 + 荧光绿

场景 废弃都市 · 废土公路 · 核电站废墟 · 地下庇护所

僵尸 Q版搞笑风格降低血腥感

特有系统

士气系统： 小队在外久了会疲劳/恐惧 → 轮换部队（变现点）

辐射区： 随机生成，需特殊单位（防化服）

尸潮： 周期性基地袭击 → 驱动防御建设

搜刮随机事件： 发现幸存者/变异体/稀有武器

✅ vs 无尽冬日： 无尽冬日 = 写实 SLG（看数字成长）| 本产品 = Q版休闲 RTS（爽感操作，3 分钟一局）

Deep Integration · Option B

星际太空深度融合

开发成本 大 蓝海标的

狱国 → 星际映射

狱国原版星际太空包装
出兵攻打城池→舰队攻击敌舰/行星前哨
兵种克制→舰船三角：护卫舰→轰炸机→航母→护卫舰
城池占领→星区/行星占领，阵营色填充
资源→能量矿石·钛合金·反物质·暗物质
联盟国战→银河阵营战：帝国 vs 联邦 vs 虫族
武将/英雄→舰长：特有技能+旗舰加成

3D 战场 · 独占优势

太空题材独有的 3D 战场维度：

  • 拉线穿越小行星带
  • 绕行行星后方进行夹击
  • 通过虫洞跳跃至战场另一端
  • 比任何地面题材都更具操作爽感

特有系统

虫洞跳跃： 舰队瞬间传送至另一区域

行星类型： 资源星/堡垒星/科技星/首都星

旗舰系统： 英雄单位，旗舰沉没则全局属性下降

空间站交易： 资源兑换·雇佣兵招募

舰船克制链： 护卫舰→轰炸机→航母→护卫舰

🌌 为什么选太空： 微信小游戏零太空策略竞品 · 「拉线=设定航线」最自然的操作映射 · 出海零文化壁垒

Deep Integration · Option C

动物王国深度融合

开发成本 大 推荐 ★★★★

核心设计

世界观 森林/草原/沙漠/冰川各动物部落的领地之争

兵种 狮子（重装）· 狼群（快速）· 大象（攻城）· 鹰（侦察）· 蛇（偷袭）

美术 顶级 Q 版卡通，电影级角色设计（参考 Zootopia/疯狂动物城）

资源 食物·水源·木材·皮毛

场景 雨林树屋 · 沙漠绿洲 · 冰原雪洞 · 火山堡垒

优势 & 风险

优势

  • 零审核风险 —— 动物主题国内最宽松
  • 全年龄段通吃，女性/儿童用户友好
  • 微信小游戏动物类有成功先例（动物餐厅）
  • 长生命周期 + 出海无文化障碍

风险

  • 核心付费群体（30-45 男）转化摩擦
  • 战争情绪被可爱稀释
  • 付费深度未验证
  • 美术量大
